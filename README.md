# LearnPlaywright - JavaScript Learning Repository

Structured practice repository for JavaScript fundamentals, syntax, operators, control flow, and basic user input programs.

## Project Structure

Current folders in this workspace:

- `Chapter_01_Basics` - Intro and setup verification
- `Chapter_02_Javascript_concepts` - JavaScript basics
- `Chapter_03_Identifier_literals` - Identifier rules, naming, comments
- `Chapter_04_Javascript_Concepts` - `var`/`let`/`const`, scope, hoisting, functions
- `Chapter_05_Literals` - String/number/null/undefined/template literals
- `Chapter_06_double_tripple_equal` - Operators and comparisons
- `Chapter_07_If_Else` - Conditional statements and logic exercises
- `Chapter_08_Switch_Statement` - Switch-case concepts and examples
- `Chapter_09_UserInput` - User input with `readline` and `prompt-sync`

## Quick Start

### Prerequisites

- Node.js 14+
- VS Code (recommended)
- Git

### Verify Setup

```bash
cd LearnPlaywright
node Chapter_01_Basics/03_JS_Verify_Setup.js
```

## How To Run Any Example

Use Node to run a file from any chapter:

```bash
node Chapter_01_Basics/01_Basics.js
node Chapter_06_double_tripple_equal/36_Comparision_Strict_loose.js
node Chapter_08_Switch_Statement/67_IQ4.js
```

## Chapter Guide

### Chapter 01 - Basics

- Intro JavaScript files and environment check script.

### Chapter 02 - JavaScript Concepts

- Core JavaScript concept refreshers.

### Chapter 03 - Identifiers and Comments

- Identifier naming rules and conventions.
- Valid vs invalid identifier examples.
- Comments and VS Code shortcut notes.

### Chapter 04 - Variables, Scope, Hoisting, Functions

- Full progression of `var`, `let`, `const` behavior.
- Function basics and hoisting examples.
- Scope differences with practical snippets.

### Chapter 05 - Literals

- Literal types in JavaScript.
- Number formats (decimal, hex, octal, binary, BigInt).
- Template literals and quote/backtick differences.

### Chapter 06 - Operators and Equality

- Arithmetic, modulus, exponentiation, logical, ternary, type operators.
- Increment/decrement and comparison examples.
- Strict (`===`) vs loose (`==`) equality practice.

### Chapter 07 - If Else

- `if`, `else if`, `else` flow.
- Real-world style decision examples.
- Practice files (even/odd, grade, leap year).

### Chapter 08 - Switch Statement

- Switch syntax, grouped cases, and default blocks.
- Break/no-break behavior.
- Interview/practice style switch questions.

### Chapter 09 - User Input

- Console input patterns in Node.js.
- `readline` examples.
- `prompt-sync` usage examples.

## Learning Path

1. Start from Chapter 01 and run each file in sequence.
2. Complete Chapters 02-04 to build JavaScript foundations.
3. Continue with Chapters 05-06 for literals and operators.
4. Practice control flow in Chapters 07-08.
5. Finish with user input programs in Chapter 09.
